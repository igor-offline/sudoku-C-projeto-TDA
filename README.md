# sudoku-C-projeto-TDA
Sudoku feito em C para a disciplina Técnicas e Desenvolvimento de Algoritmo

## Descrição

Jogo de Sudoku feito em linguagem C, rodando no terminal.

Feito para praticar lógica de programação e manipulação de matrizes

Gera um tabuleiro aleatório, permite jogar e valida as regras.

## Compilar usando GCC:
gcc sudoku.c -o sudoku

E executar:

./sudoku

## Como jogar
Digite linha e coluna (1 a 9)

Insira o número desejado (1 a 9)

## Regras do jogo
Números iguais não podem existir na mesma posição, linha, coluna e/ou bloco 3x3

O jogo termina quando o tabuleiro estiver completo, com cada linha e coluna com números de 1 a 9 sem repetição
